# Installation

To start using Artisanal in your Dart project, add it to your `pubspec.yaml`:

```yaml
dependencies:
  artisanal:
    path: ../path/to/artisanal # Or use the pub.dev version once published
```

Then run:

```bash
dart pub get
```

## Requirements

- **Dart SDK**: ^3.0.0
- **Terminal**: A modern terminal with ANSI support (e.g., iTerm2, Alacritty, Kitty, Windows Terminal).
