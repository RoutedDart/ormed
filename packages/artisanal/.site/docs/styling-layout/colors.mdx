# Colors

Artisanal provides a robust color system that supports everything from basic 16-color ANSI to full 24-bit TrueColor.

## Color Profiles

Artisanal can automatically detect the terminal's color capabilities and downgrade colors if necessary.

- **TrueColor**: 16.7 million colors.
- **ANSI256**: 256 colors.
- **ANSI**: 16 colors.
- **NoColor**: Plain text only.
