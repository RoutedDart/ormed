# File Picker

The `FilePicker` bubble provides a terminal-based file and directory browser.

## Basic Usage

```dart file=../../../example/tui/examples/file-picker/main.dart#file_picker_usage
```

## Features

- **Navigation**: Browse through directories and select files.
- **Filtering**: Filter files by extension or name.
- **Custom Icons**: Support for file-type specific icons.
- **Permissions**: Displays file permissions and sizes.

## Example

See the [file_picker_demo.dart](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/tui/file_picker_demo.dart) for a complete example.
