# Quick Start

Let's build a simple counter application to see how Artisanal works.

## 1. Define the Model

The model represents the state of your application.

```dart file=../../../example/tui/counter.dart#model_example
```

## 2. Run the Program

Use the `runProgram` helper to start the TUI loop.

```dart file=../../../example/tui/counter.dart#program_example
```

## 3. Run the Application

```bash
dart run bin/main.dart
```
