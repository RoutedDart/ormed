# View

The `view` function is responsible for rendering your `Model` into a visual representation. In Artisanal, this is typically a `String` that will be printed to the terminal.

## The View Signature

```dart
String view()
```

## Rendering Strings

The simplest view returns a plain string:

```dart file=../../../example/tui/counter.dart#view_example
```

## Using Styles

You can use the `Style` API to add colors, borders, and padding to your view:

```dart file=../../../example/doc_examples.dart#styled_view_example
```

## Layout Primitives

Artisanal provides layout primitives like `joinHorizontal` and `joinVertical` to combine multiple strings:

```dart file=../../../example/doc_examples.dart#layout_view_example
```

## Advanced Rendering (UV)

For more complex UIs, you can use the **Ultraviolet (UV)** renderer, which provides a cell-buffer based approach with layers and compositing. See the [Advanced Rendering](../advanced-rendering/overview) section for more details.
