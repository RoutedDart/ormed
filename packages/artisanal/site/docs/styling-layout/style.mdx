# Style

The `Style` class provides a fluent, chainable API for styling terminal text. It is heavily inspired by the Go `lipgloss` library.

## Basic Usage

You can create a style and apply it to a string using the `render` method.

```dart file=../../../example/fluent_style_example.dart#basic_style
```

## Chainable Methods

The `Style` class supports a wide range of methods for customizing text appearance:

- **Text Attributes**: `bold()`, `italic()`, `underline()`, `strikethrough()`, `dim()`, `inverse()`, `blink()`.
- **Colors**: `foreground(Color)`, `background(Color)`, `underlineColor(Color)`.
- **Spacing**: `padding(top, [right, bottom, left])`, `margin(top, [right, bottom, left])`.
- **Dimensions**: `width(int)`, `height(int)`, `maxWidth(int)`, `maxHeight(int)`.
- **Layout**: `align(HorizontalAlign)`, `alignVertical(VerticalAlign)`, `border(Border)`.

## Inheritance

Styles can inherit properties from other styles using the `inherit` method. This is useful for creating base styles and specializing them.

```dart file=../../../example/fluent_style_example.dart#inheritance_example
```
