# Colors

Artisanal provides a robust color system that supports everything from basic 16-color ANSI to full 24-bit TrueColor.

## The Color Class

The `Color` class is the base for all colors in Artisanal.

### Basic Colors (16 colors)

```dart
final red = BasicColor.red;
final brightBlue = BasicColor.brightBlue;
```

### ANSI Colors (256 colors)

```dart
final orange = AnsiColor(208);
```

### TrueColor (RGB)

```dart
final custom = Color.rgb(255, 100, 50);
final hex = Color.hex('#ff6432');
```

## Color Profiles

Artisanal can automatically detect the terminal's color capabilities and downgrade colors if necessary.

- **TrueColor**: 16.7 million colors.
- **ANSI256**: 256 colors.
- **ANSI**: 16 colors.
- **NoColor**: Plain text only.

## Adaptive Colors

`AdaptiveColor` allows you to define different colors for light and dark terminal backgrounds.

```dart
final text = AdaptiveColor(
  light: Color.black,
  dark: Color.white,
);
```
