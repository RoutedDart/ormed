# Writer API

The Writer API provides a set of global functions for printing styled text to the terminal, similar to Go's `fmt` package.

## Print Functions

- `Print(Object? arg)`: Prints a styled object.
- `Println(Object? arg)`: Prints a styled object followed by a newline.
- `Printf(String format, [List<Object?>? args])`: Prints a formatted string.

## Sprint Functions

The `Sprint` family of functions returns the styled string instead of printing it.

- `Sprint(Object? arg)`
- `Sprintln(Object? arg)`
- `Sprintf(String format, [List<Object?>? args])`

## Usage

```dart
import 'package:artisanal/style.dart';

Println(Style().bold().render("Important Message"));

final msg = Sprintf("User %s logged in", ["admin"]);
Println(Style().foreground(Colors.green).render(msg));
```
