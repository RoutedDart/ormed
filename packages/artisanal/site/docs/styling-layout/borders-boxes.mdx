# Borders & Boxes

Artisanal allows you to wrap styled content in various border styles to create boxes and panels.

## Predefined Borders

The `Border` class provides several predefined border styles:

- `Border.normal`: Standard single-line border.
- `Border.rounded`: Single-line border with rounded corners.
- `Border.thick`: Double-line or thick border.
- `Border.hidden`: No visible border, but still occupies space for padding/margin.

## Applying Borders

You can apply a border to a `Style`.

```dart
final style = Style()
  .border(Border.rounded)
  .borderColor(Colors.blue)
  .padding(1);

print(style.render("This is inside a box!"));
```

## Custom Borders

You can also define custom borders by specifying the characters for each side and corner.

```dart
final customBorder = Border(
  top: '-',
  bottom: '-',
  left: '|',
  right: '|',
  topLeft: '+',
  topRight: '+',
  bottomLeft: '+',
  bottomRight: '+',
);
```
