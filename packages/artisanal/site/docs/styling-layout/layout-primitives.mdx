# Layout Primitives

Artisanal provides several primitives for combining and aligning multiple styled strings.

## Joining Strings

### `joinHorizontal`

Joins multiple strings horizontally. You can specify the vertical alignment of the strings.

```dart file=../../../example/fluent_style_example.dart#layout_horizontal
```

### `joinVertical`

Joins multiple strings vertically. You can specify the horizontal alignment of the strings.

```dart file=../../../example/fluent_style_example.dart#layout_vertical
```

## Alignment

The `align` and `alignVertical` methods on `Style` allow you to position text within a fixed-size box.

```dart file=../../../example/doc_examples.dart#alignment_usage_example
```

## Stacking

The `stack` function allows you to overlay multiple strings on top of each other (useful for backgrounds or watermarks).

```dart file=../../../example/doc_examples.dart#stack_example
```
