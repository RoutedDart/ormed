# Themes

Artisanal includes a semantic theming system that allows you to define a consistent look and feel for your application and easily switch between different color palettes.

## ThemePalette

The `ThemePalette` class defines a set of semantic color roles:

- `primary`: The main brand color.
- `secondary`: Accent color.
- `background`: Main background color.
- `foreground`: Main text color.
- `success`, `warning`, `error`, `info`: Status colors.

## Built-in Themes

Artisanal comes with several built-in themes:

- `ThemePalette.nord`
- `ThemePalette.dracula`
- `ThemePalette.solarizedDark` / `ThemePalette.solarizedLight`
- `ThemePalette.gruvboxDark` / `ThemePalette.gruvboxLight`
- `ThemePalette.catppuccin`
- `ThemePalette.tokyoNight`
- `ThemePalette.oneDark`

## Using Themes

You can use the theme colors in your styles:

```dart
final theme = ThemePalette.nord;

final titleStyle = Style()
    .foreground(theme.primary)
    .background(theme.background)
    .bold();

final errorStyle = Style().foreground(theme.error);
```

## Adaptive Colors

Themes use `AdaptiveColor`, which can automatically switch between light and dark variants based on the terminal's color profile.

```dart
final adaptiveColor = AdaptiveColor(
  light: Color.black,
  dark: Color.white,
);
```

## Custom Themes

You can create your own `ThemePalette`:

```dart
final myTheme = ThemePalette(
  primary: Color.hex('#ff00ff'),
  secondary: Color.hex('#00ffff'),
  background: Color.black,
  foreground: Color.white,
  // ...
);
```
