# Themes

Artisanal includes a semantic theming system that allows you to define a consistent look and feel for your application and easily switch between different color palettes.

## ThemePalette

The `ThemePalette` class defines a set of semantic color roles:

- `primary`: The main brand color.
- `secondary`: Accent color.
- `background`: Main background color.
- `foreground`: Main text color.
- `success`, `warning`, `error`, `info`: Status colors.

## Built-in Themes

Artisanal comes with several built-in themes:

- `ThemePalette.nord`
- `ThemePalette.dracula`
- `ThemePalette.solarizedDark` / `ThemePalette.solarizedLight`
- `ThemePalette.gruvboxDark` / `ThemePalette.gruvboxLight`
- `ThemePalette.catppuccin`
- `ThemePalette.tokyoNight`
- `ThemePalette.oneDark`
