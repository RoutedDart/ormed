# Examples

The `artisanal` package includes a wide variety of examples demonstrating different features and use cases. You can find them in the `example/` directory of the repository.

## Running Examples

To run an example, use the `dart run` command from the root of the `artisanal` package:

```bash
cd packages/artisanal
dart run example/tui/counter.dart
```

## Featured Examples

### Interactive TUIs
- **[Counter](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/tui/counter.dart)**: A simple counter demonstrating TEA.
- **[Command Center](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/command_center_demo.dart)**: A complex dashboard with multiple panels, themes, and sparklines.
- **[Trello](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/tui/examples/trello.dart)**: A Kanban board with drag-and-drop support.

### Styling & Layout
- **[Fluent Style](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/fluent_style_example.dart)**: Demonstrates the chainable Style API.
- **[Layout](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/lipgloss_layout.dart)**: Shows how to compose complex layouts.

### Advanced Rendering
- **[UV Demo](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/uv_demo.dart)**: Demonstrates the high-performance Ultraviolet renderer.
- **[Kitty Graphics](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/kitty_image_demo.dart)**: Shows terminal graphics support.
