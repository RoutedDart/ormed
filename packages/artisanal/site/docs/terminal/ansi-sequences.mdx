# ANSI Sequences

The `Ansi` class provides a comprehensive collection of ANSI escape sequence constants and helper methods.

## Common Sequences

- `Ansi.reset`: Resets all text attributes.
- `Ansi.bold`, `Ansi.italic`, `Ansi.underline`.
- `Ansi.clearScreen`, `Ansi.clearLine`.
- `Ansi.hideCursor`, `Ansi.showCursor`.

## Cursor Movement

`Ansi` includes methods for generating cursor movement sequences:

- `Ansi.cursorTo(int x, int y)`
- `Ansi.cursorUp(int n)`
- `Ansi.cursorForward(int n)`

## Colors

While the `Style` API is recommended for most use cases, you can generate raw ANSI color sequences:

- `Ansi.fg(Color color)`
- `Ansi.bg(Color color)`

## Usage

```dart
import 'package:artisanal/terminal.dart';
import 'dart:io';

stdout.write(Ansi.cursorTo(0, 0));
stdout.write(Ansi.bold);
stdout.write('Top Left Bold');
stdout.write(Ansi.reset);
```
