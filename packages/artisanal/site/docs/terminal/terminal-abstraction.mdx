# Terminal Abstraction

Artisanal provides a unified `Terminal` interface that abstracts away the differences between various terminal environments and I/O sinks.

## The `Terminal` Interface

The `Terminal` interface defines common operations:

- `write(String s)` / `writeln(String s)`
- `hideCursor()` / `showCursor()`
- `clearScreen()`
- `enableRawMode()` / `disableRawMode()`
- `getSize()`: Returns the current terminal width and height.

## Implementations

- **`StdioTerminal`**: The standard implementation using `dart:io`'s `stdout` and `stdin`.
- **`StringTerminal`**: A virtual terminal that captures output into a string, useful for unit testing.
- **`TtyTerminal`**: A specialized terminal that uses `/dev/tty` for control and input, allowing output to be redirected while maintaining interactivity.

## Usage

```dart
import 'package:artisanal/terminal.dart';

final terminal = StdioTerminal();
terminal.hideCursor();
terminal.write(Ansi.clearScreen);
// ...
terminal.showCursor();
```
