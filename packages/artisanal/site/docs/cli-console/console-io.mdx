# Console & I/O

The `Console` class is the primary entry point for high-level terminal I/O in Artisanal. It provides a rich API for writing to the terminal, handling verbosity, and running tasks with status indicators.

## Basic Output

You can use `Console` to write styled text to the terminal.

```dart
import 'package:artisanal/artisanal.dart';

final console = Console();

console.write('Hello, ');
console.writeln('Artisanal!', style: Style().bold().foreground(Colors.green));
```

## Task Indicators

`Console` can run asynchronous tasks and display a spinner or status indicator while they are running.

```dart
await console.task('Downloading data...', () async {
  await Future.delayed(Duration(seconds: 2));
  return TaskResult.success('Done!');
});
```

## Interactive Prompts

The `Components` class (accessible via `console.components`) provides interactive prompts for user input.

```dart
final name = await console.components.ask('What is your name?');
final confirmed = await console.components.confirm('Are you sure?');
```
