# Progress & Spinner

Artisanal provides components for visual feedback during long-running operations.

## Spinner

The `Spinner` bubble displays an animated loading indicator.

```dart
import 'package:artisanal/tui.dart';
import 'package:artisanal/bubbles.dart';

class MyModel implements Model {
  SpinnerModel spinner = SpinnerModel(spinner: Spinners.dot);

  @override
  Cmd? init() => spinner.init();

  @override
  (Model, Cmd?) update(Msg msg) {
    final (newSpinner, cmd) = spinner.update(msg);
    spinner = newSpinner;
    return (this, cmd);
  }

  @override
  String view() => '${spinner.view()} Loading...';
}
```

### Predefined Spinners

Artisanal includes many predefined spinner animations in the `Spinners` class:
- `Spinners.line`
- `Spinners.dot`
- `Spinners.miniDot`
- `Spinners.jump`
- `Spinners.pulse`
- `Spinners.globe`
- `Spinners.moon`
- `Spinners.monkey`

## Progress Bar

The `Progress` bubble displays a horizontal progress bar.

```dart
import 'package:artisanal/tui.dart';
import 'package:artisanal/bubbles.dart';

class MyModel implements Model {
  ProgressModel progress = ProgressModel();

  @override
  Cmd? init() => null;

  @override
  (Model, Cmd?) update(Msg msg) {
    // Update progress percentage (0.0 to 1.0)
    if (msg is ProgressMsg) {
      return (this, progress.setPercent(msg.percent));
    }
    
    final (newProgress, cmd) = progress.update(msg);
    progress = newProgress;
    return (this, cmd);
  }

  @override
  String view() => progress.view();
}
```

### Features
- **Animated Transitions**: Smoothly animates between progress values.
- **Custom Colors**: Set different colors for the filled and empty parts.
- **Width Control**: Easily adjust the width of the bar.
