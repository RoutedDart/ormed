# Viewport

The `Viewport` bubble provides a scrollable window into a larger piece of content. It's essential for displaying long text, logs, or large data sets in a limited terminal area.

## Basic Usage

```dart
import 'package:artisanal/tui.dart';
import 'package:artisanal/bubbles.dart';

class MyModel implements Model {
  late ViewportModel viewport;

  MyModel() {
    viewport = ViewportModel(width: 30, height: 10);
    viewport.setContent('Line 1\nLine 2\n...\nLine 100');
  }

  @override
  Cmd? init() => null;

  @override
  (Model, Cmd?) update(Msg msg) {
    final (newViewport, cmd) = viewport.update(msg);
    viewport = newViewport;
    return (this, cmd);
  }

  @override
  String view() => viewport.view();
}
```

## Features

- **Scrolling**: Supports vertical and horizontal scrolling.
- **Key Bindings**: Built-in support for arrow keys, PageUp/PageDown, Home/End.
- **Mouse Support**: Can be configured to respond to mouse wheel events.
- **Syncing**: Can be synced with other viewports or components.
- **Custom Styles**: Style the viewport borders and scroll indicators.
