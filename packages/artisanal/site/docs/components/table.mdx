# Table

The `Table` bubble provides an interactive data grid with support for columns, rows, selection, and custom styling.

## Basic Usage

```dart file=../../../example/tui/examples/table/main.dart#table_usage
```

## Features

- **Selection**: Navigate through rows using arrow keys.
- **Custom Rendering**: Define how cells are rendered based on their content.
- **Styling**: Customize headers, borders, and selected row styles.
- **Dynamic Data**: Easily add, remove, or update rows and columns.

## Example

See the [table_demo.dart](https://github.com/kingwill101/ormed/blob/main/packages/artisanal/example/tui/table_demo.dart) for a complete example.
