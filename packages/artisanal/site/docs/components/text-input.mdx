# Text Input

The `TextInput` bubble provides a single-line text entry field with support for placeholders, focus management, and custom styling.

## Basic Usage

```dart
import 'package:artisanal/tui.dart';
import 'package:artisanal/bubbles.dart';

class MyModel implements Model {
  TextInputModel input = TextInputModel(placeholder: 'Enter your name...');

  @override
  Cmd? init() => TextInputModel.focus(input);

  @override
  (Model, Cmd?) update(Msg msg) {
    final (newInput, cmd) = input.update(msg);
    input = newInput;
    return (this, cmd);
  }

  @override
  String view() => 'Name: ${input.view()}';
}
```

## Features

- **Focus Management**: Use `TextInputModel.focus()` and `TextInputModel.blur()`.
- **Placeholders**: Display a hint when the input is empty.
- **Password Mode**: Mask input characters for sensitive data.
- **Validation**: Attach validation functions to provide feedback.
- **Custom Styling**: Change the style of the text, placeholder, and cursor.
