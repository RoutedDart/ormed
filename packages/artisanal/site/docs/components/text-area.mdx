# Text Area

The `TextArea` bubble provides a multi-line text entry field with support for line numbers, focus management, and custom styling.

## Basic Usage

```dart file=../../../example/tui/examples/textarea/main.dart#textarea_usage
```

## Features

- **Multi-line Editing**: Handles multiple lines of text with automatic scrolling.
- **Line Numbers**: Optional line numbers for better orientation.
- **Focus Management**: Use `TextAreaModel.focus()` and `TextAreaModel.blur()`.
- **Placeholders**: Display a hint when the area is empty.
- **Custom Styling**: Change the style of the text, placeholder, and cursor.
